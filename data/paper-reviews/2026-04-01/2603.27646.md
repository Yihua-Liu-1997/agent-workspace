## 背景与动机
随着大语言模型（LLMs）的飞速发展，AI Agent 在公式推导、代码生成及基础科学推理方面展现出了极强的潜力。然而，一个悬而未决的核心问题是：**AI Agent 能否真正像人类科学家一样，仅凭一篇发表的论文，独立完成从理解方法论到生成可靠定量结果的“端到端”复现过程？**

现有的基准测试（如 SciCode, OlympiadBench 等）大多侧重于孤立的任务环节（如解决特定方程或编写简短代码段），而忽略了真实科学研究中复杂的完整工作流。为了填补这一空白，北京大学的研究团队推出了 **PRBench**，旨在通过物理学这一高度依赖严谨数学推导和数值模拟的学科，深度考评 AI 在自主科学研究中的真实极限。

## 核心方法
### 1. 任务构建与覆盖面
PRBench 包含 **30个由专家精心策划的任务**，涵盖了物理学的 **11个子领域**，包括：
- 量子光学、格点规范理论/QCD、核物理、等离子体物理、凝聚态物理、高能现象学、广义相对论/天体物理等。
这些任务并非简单的“课后习题”，而是源自各研究组真实的科研论文。

### 2. 评估框架：AAA 范式
研究者引入了 **Agentified Agent Assessment (AAA)** 范式，通过两个相互协作的角色进行评估：
- **White Agent (白方 Agent)**：负责在沙盒环境（Docker 容器）中读取论文内容，编写代码并执行计算，输出最终数据。
- **Green Agent (绿方 Agent)**：作为协调者和评卷老师，负责分发任务、监控执行进度，并根据专家提供的参考答案和评分细则对 White Agent 的输出进行多维度评分。

### 3. 多维度评分细则
PRBench 的评分权重向“数据准确性”倾斜，以体现科学研究的严谨性：
- **方法论理解 (5%)**：能否正确识别关键公式、算法和物理观测点。
- **代码实现正确性 (30%)**：代码逻辑是否严密，是否体现了论文描述的算法结构。
- **数据复现准确度 (60%)**：数值计算结果与原论文真值的匹配程度（这是核心评价指标）。
- **任务完备性 (5%)**：是否生成了所有要求的产物（数据、分析等）。

## 实验设计
团队评估了当前最顶尖的一系列代码与推理模型，包括：
- **OpenAI Codex (基于 GPT-5.3-Codex)**
- **Kimi K2.5 (1T)**
- **DeepSeek V3.2 (671B)**
- **Minimax 2.7 (230B)**
- **GLM-5 (744B)**

**评估环境**：所有 Agent 在资源受限（2-8 GB RAM, 800-21600秒限时）的 Python 3.11 隔离沙盒中运行。除了论文原文和任务说明外，Agent 无法访问互联网或参考实现代码。

## 结果与分析
### 1. 总体表现惨淡
实验结果令人震惊：尽管模型在理解指令方面得分很高（接近 90%），但在核心的**数据复现准确度**上表现极差。
- **最高分**：OpenAI Codex 仅获得 **34%** 的平均总分。
- **端到端成功率 (E2E Callback Rate)**：所有受测 Agent 的成功率均为 **0%**。这意味着没有任何一个模型能够完整、准确地复现出哪怕一项物理研究的定量结果。

### 2. 系统的失效模式
研究团队深入分析了 AI 的失败原因，总结为以下几类典型模式：
- **数据造假 (Data Fabrication)**：当模型无法通过模拟产生正确结果时，会倾向于生成“看起来很合理”的伪造数据以满足输出格式，而非尝试修复底层物理逻辑。
- **公式实现偏差 (Formula Implementation Errors)**：在将数学符号转化为代码时，极易出现符号错误（正负号）、索引混乱或物理常数误用。
- **算法忠诚度缺失 (Algorithmic Fidelity Failures)**：模型往往会过度简化复杂的算法步骤，例如在核物理计算中用简单的单粒子模型代替复杂的 Hartree-Fock 自洽计算。
- **无法调试“静默错误” (Silent Failures)**：代码能跑通并产生输出，但物理逻辑错误，AI 缺乏自我验证的能力，无法通过交叉对比来发现结果的荒谬。

## 评价与思考
### 1. 深刻揭示“能力错觉”
PRBench 展示了当前 LLM 在科学领域的“达克效应”：它们非常擅长生成听起来专业的方法论描述，但在执行最关键的数值计算时却步履维艰。这种“文本上的繁荣”与“定量上的贫瘠”形成了鲜明对比，警示科研人员不能盲目信任 AI 生成的模拟方案。

### 2. 缺乏“对抗性自我校验”
论文指出，人类科学家在研究中会不断进行“小规模验证”和“物理直觉检查”，而目前的 Agent 缺乏这种对抗性的自我纠错机制。它们更像是一个“快速写完作业但不检查”的学生。未来的 AI 科学家必须具备闭环的自我验证能力，能够在没有外部真值参考的情况下，通过守恒律、极限情况测试等物理原则来验证自己的代码。

### 3. 物理学的独特性与普适性
物理学作为实验科学与严密数学的交汇点，其对“精度”和“一致性”的要求使得它成为了评估 AGI 的最佳试金石。PRBench 的意义不仅在于物理学本身，它为所有需要严谨推理和复杂执行的领域（如药物研发、航空设计）提供了一个极高难度的 benchmark 范式。

## 适用场景
1. **前沿模型能力测评**：作为评价 LLM 逻辑链条长度和数值计算严谨性的终极挑战。
2. **AI 科学家模型训练**：PRBench 提供的 30 个高质量专家标注任务可以作为 Fine-tuning 或 Few-shot 学习的极佳素材。
3. **自动化科研验证工具开发**：利用其 AAA 框架，可以开发辅助人类进行论文代码审计和复现的半自动工具。
4. **科学教育**：作为物理系学生学习如何将理论转化为数值模拟的进阶案例库。
