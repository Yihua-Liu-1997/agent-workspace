assistant: **Brave Search 工作原理：**

1. **自建搜索引擎** — 有自己的爬虫、索引、排名算法（不依赖 Google/Bing）
2. **Summarizer AI** — 搜索结果用自家的 LLM 总结，不是 OpenAI 的
3. **实时数据** — 直接从网络获取，不是预训练的知识库
4. **透明度** — 会显示信息来源链接

---

**和 Google 的区别：**

| | Brave | Google |
|---|---|---|
| **核心技术** | 自家 LLM + 爬虫 | PageRank + 传统算法 |
| **AI 总结** | ✅ Summarizer 直接给答案 | ⚠️ 也有但不如 Brave 直接 |
| **数据来源** | 实时网络 | 实时网络 |
| **隐私** | ⭐⭐⭐⭐⭐ 不追踪 | ⭐⭐ 会追踪 |

---

**简单说：Brave 是"搜索引擎 + AI 总结"，独立于 Google/Bing，隐私友好。**
