# 记忆整理指南

每天 23:00 记忆时遵循此规则。

## 分层记忆系统

```
memory/YYYY-MM-DD.md  ← 所有事情（今天的对话、完成的任务、临时想法）
        ↓
    每天记忆时筛选
        ↓
MEMORY.md            ← 只放"真正重要"的东西
```

## 3 个判断问题

什么进 MEMORY.md？用这 3 个问题判断：

1. **这会影响未来的决策吗？** → 是 → MEMORY.md
2. **这是关于你的核心身份/偏好的吗？** → 是 → MEMORY.md
3. **半年后还记得这件事会有用吗？** → 是 → MEMORY.md

**全部"否" → 只放 `memory/YYYY-MM-DD.md`，不要放 MEMORY.md**

## MEMORY.md 适合放

- 你的核心需求（每天要论文推送）
- 你的偏好（喜欢实用的东西）
- 重要关系（你叫我"宝宝"）
- 关键决定（cron 改成提醒模式）
- 长期有用的教训

## MEMORY.md 不适合放

- 今天修了什么 bug
- 具体的命令/代码（放 TOOLS.md 或脚本注释）
- 临时想法
- 日常琐事

## 每天 23:00 记忆流程

1. 回顾今天的 `memory/YYYY-MM-DD.md`
2. 用 3 个问题筛选
3. 只把通过筛选的提炼到 MEMORY.md
4. 原始记录留在当天文件，不要删除
